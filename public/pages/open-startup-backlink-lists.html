Why this exists

We compiled this data from directory submission sources with attribution. Now we are open-sourcing it to help the startup community.

  • Source-aware Entries are aggregated from multiple public lists with references.
  • Maintained conservatively We occasionally add more to the lists when a site makes sense or when it comes through community recommendations.
  • Built to be useful The goal is a cleaner starting point for research, not a promise that every resource will fit every launch.

How to use it

A simple workflow for founders and marketers who want the page to be practical instead of over-explained.

  • Start with the live explorers Use the directories explorer when you want one working chart with adjustable type filters, then move to guest posts or narrower landing pages if needed.
  • Export only when needed Pull the JSON files if you want to sort, enrich, or integrate the data into your own workflow.
  • Contribute selectively If you find a strong addition, submit it through the repo rather than treating the list as a volume game.
